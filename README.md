# Простой чат на Nuxt.js 3

Этот проект представляет собой простой чат, разработанный с использованием Nuxt.js 3 и Composition API.

## Описание
Этот проект является демонстрацией создания базового чат-приложения с использованием последних возможностей Nuxt.js 3, включая Composition API для организации логики компонентов. Проект нацелен на простоту и ясность, чтобы служить отправной точкой для изучения Nuxt 3.

## Технологии
*   Nuxt.js 3
*   Composition API
*   Vite (для быстрой разработки и сборки)
*   TypeScript
*   TailwindCSS (для стилизации)

## Установка
Клонируйте репозиторий:
```bash
git clone git@github.com:ShafikovVA/chat-vue-app.git
cd chat-vue-app
```
Установите зависимости:
```bash
bun install
```

## Доступные скрипты
В корне проекта вы можете запускать следующие команды:

*   `bun run dev`: Запускает приложение в режиме разработки с горячей перезагрузкой.
*   `bun run build`: Собирает приложение для продакшена в директорию `.output`.
*   `bun run preview`: Локально запускает собранное приложение для предпросмотра.

## Запуск в режиме разработки
```bash
bun run dev
```
Откройте `http://localhost:3000` в браузере.

## Сборка для продакшена
```bash
bun run build
```
Эта команда собирает приложение для продакшена в директорию `.output`.

## Предварительный просмотр сборки
```bash
bun run preview
```
